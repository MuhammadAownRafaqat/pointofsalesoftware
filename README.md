# Django POS System

Simple Point of Sale system built with Django. Watch this video and see how to run it and make it: https://youtu.be/A4HoHm6qKJQ?si=lgtDv2kFF-J6KKt5

## Features
- Product & Category management
- Inventory tracking
- Sales recording
- Django Admin backend
- Simple POS interface

## Setup
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
